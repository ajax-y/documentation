### [2] Rock Paper Scissor

I build a simple text based rock paper scissor game which the game runs between the user and the computer. The user can pick the **rock**, **paper**, **scissor** by choosing the number between 1 and 3. 1 denotes as **rock** , 2 denotes as **paper** and 3 denotes as **scissor**. For computerChoice I used random number generator in C to generate the number between 1 and 3. Based on that the game result will be displayed.

Check out the [Rock Paper Scissor](rock-paper-scissor/README.md) for more details.

### [3] Hangman

I build the text based hangman game which we need to predict the words. The words first letter is given and remaining are given as underscore. We need to find each letters. If you guess wrong, for each wrong letter the lives will be decreased by 1. Totally 3 lives will be given, if your lives comes to zero the game will be lose.

Check out the [Hangman](hangman/README.md) for more details.
