## 🐚 Linux Commands I've Mastered
Here are the essential commands I learned while solving labs and exploring the system on my own:

### 📁 File & Directory Navigation
* `pwd`: Prints the current working directory path.
* `ls -la`: Lists all files, including hidden ones, with details like permissions.
* `cd`: Changes directories. Use `cd ..` to go back one step.

### 🔍 Searching & Filtering Text
* `cat`: Displays the full content of a file in the terminal.
* `grep`: Searches for specific keywords inside files or output.
* `find / -name "flag.txt"`: Searches the entire system for a specific file.

### 🛡️ Permissions & System Info
* `chmod`: Changes file permissions (e.g., `chmod +x script.sh` to make it runnable).
* `whoami`: Displays the username of the current active session.
* `ssh username@host`: Connects securely to remote CTF servers (like OverTheWire).
