# What is Breach Lab

**Breach Lab** is a website where peoples can learn **Cyber Security** by doing **CTF (Capture The Flag)**.

- **Breach Lab** is a persistent server with real vulnerabilities Linux boxes you SSH into and root it.
- Free, No Signup, No Paywall, No Certificate.

# What is not Breach Lab
- Not a Multiple choice questions.
- Not a curriculum, not a course, not a bootcamp.
- Flags here are real shell passwords, in real machine.

## Who this for
Operators. People who would rather get stuck for six hours and figure something out than be told the answer in three minutes. People who understand that the only way you actually learn this craft is by doing it on systems that fight back.
