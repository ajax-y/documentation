Rock Paper Scissor

The program main idea is to play the rock paper scissor game in the text based. Because I still don't know how can I make the graphical based games. The game is between the user and the computer. The user and computer picks the options and they are compared and shown the output.

Library

In this program I used three library, namely stdio, stdlib, and time. We all used stdio for input and output operations. stdlib and time is used for generating a random number which we could use it as a computer option.

Welcome

First when user runs the program they need to know what the program does, and what are the inputs we need to give to it. For that purpose I created the welcome page to show the user,

        Rock Paper Scissor
##################################

[1] Play Game
[2] Exit

Enter your option:

This shows the program is for rock paper scissor game. So the user got two options. Whether they can play the game or exit.

Option 1 (Play Game)

If they choose the option one the game shows its interface. It ask the user to enter the input.

        Rock Paper Scissor
##################################

[1] Rock
[2] Paper
[3] Scissor
[4] Exit

Enter your option:

From here the user can enter their input. When they select their input it stored in the variable named userChoice.

Here the user don't have to type full word. They just type the numbers. Because I consider the option 1 as rock, 2 as paper, and 3 as scissor. Because I thought it's the best way for doing.

Now we need the computer option. For that purpose I use random to generate a number between 1 and 3, and used it as a computerChoice.

Option 2 (Exit)

When the user enter option 2, the program exit.

Error

When the user enter number which is less than 1 and greater than 2, the program shows an error. 

Please enter a valid option

Info

For the computerChoice I used a random number which was generated by the program itself. In python I know how can I generate the random number. But in C I don't know how could I generate the random number. For that purpose I used the google to know how could I can able to generate the random number.

/*

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    int option, nextoption, userChoice, computerChoice;
    option = 0; //for loop bypass
    //For generate a random number which is the computerChoice
    srand(time(NULL));
    while (1==1){
        if (option !=1){
            // Wecome message
            printf("\tRock Paper Scissor\n");
            printf("##################################\n\n");
            printf("[1] Play Game\n");
            printf("[2] Exit\n\n");
            printf("Enter your option: ");
            scanf("%d", &option);
        }
        // Condition to exit from the game or play the game
        if (option >2 || option <1){
            printf("Please enter a valid option.");
            option = 0; //Reset to show the wecome message again
        }
        // For exit from the program
        else if(option == 2){
            printf("Thank You! Come again.");
            break;
        }
        // For valid option
        else{
            // Game interface
            printf("\tRock Paper Scissor\n");
            printf("##################################\n\n");
            printf("[1] Rock\n");
            printf("[2] Paper\n");
            printf("[3] Scissor\n\n");
            printf("Enter your option: ");
            // Getting user choice
            scanf("%d", &userChoice);
            // Generating computer choice
            computerChoice = (rand()%3) + 1;

            // Comparing the userChoice and computerChoice
            if (userChoice >3 || userChoice <1){
                // For a non valid inputs
                printf("Enter a valid option.");
            }
            else{
                // Reveal the choices to the player
                printf("\nYou chose: %s", (userChoice == 1) ? "Rock" : (userChoice == 2) ? "Paper" : "Scissor");
                printf("\nComputer chose: %s\n", (computerChoice == 1) ? "Rock" : (computerChoice == 2) ? "Paper" : "Scissor");
                //For a valid inputs
                if (userChoice == computerChoice){
                    // Tie game
                    printf("\n\t[The game is TIE]\n");
                }
                else if((userChoice == 1 && computerChoice == 3) || (userChoice == 2 && computerChoice == 1) || (userChoice == 3 && computerChoice == 2)){
                    // For user winning
                    printf("\n\t[You WON the game]\n");
                }
                else{
                    // FOr computer win or user lose
                    printf("\n\t[You LOSE the game]\n");
                }
                // Play again menu
                printf("\n\tRock Paper Scissor\n");
                printf("##################################\n\n");
                printf("[1] Play again\n");
                printf("[2] Exit\n");
                printf("Enter your option: ");
                scanf("%d", &nextoption);
                if (nextoption >2 || nextoption <1){
                    printf("Enter a valid option.");
                    option =0; // Reset to show the welcome message again
                }
                else if(nextoption == 2){
                    printf("Thank You! Come Again.");
                    break;
                }
                else {
                    option =1;
                    printf("\n");
                }
            }
        }
    }
}
